HTML

Abreviação para a expressão inglesa Hyper Text Markup Language, que significa: "Linguagem de marcação de hipertexto", é uma linguagem de marcação utilizada na contrução de páginas na web. Documentos html podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

Estrutura do HTML

Um documento HTML começa com a tag html e termina com /html. Ele também contém um cabeçalho que descreve o título da página e um corpo no qual se encontra o conteúdo da página. O cabeçalho é delimitado pelas tags head e /head. Já o corpo é delimitado pelas tags body e /body.

Exemplo:

      <html>
        <head>
          <title>Título da página</title>
        </head>
        <body>
          Conteúdo da página
        </body>
      </html>

Principais TAGS

HTML (<html></html>)

É a tag de construção para o HTML. Se você está criando um arquivo html, você tem que usar para incorporar todo o seu conteúdo dentro da tag HTML.

DOCTYPE (< !DOCTYPE html>)

É usado para definir o tipo de documento. Não é uma tag html. É declarado antes da tag HTML, pois é a instrução para o site que diz que o documento à frete é do tipo HTML.

HEAD (<head></head>)

Depois de abrir a tag HTML, a tag head deve ser a primeira coisa a vir. É a tag que não contém nenhum texto, mas engloba todas as outras tags. Título, estilo, meta, etc, todas as tags estão escritas dentro dele. É usado para descrever o conteúdo do site.

TITLE (<title>This is heading.</title>)

Essa tag é usada quanbdo você deseja dar um titúlo ao seu conteúdo. A tag titúlo também vem sob a tag head. Não inclui nenhuma outra etiqueta dentro dela. Somente texto pode ser escrito dentro dessa tag.

BODY (<body>This is the body.</body>)

Qualquer coisa que você vê no site está escrito sob esta tag. Portanto, o conteúdo que você deseja deve ser visível para as pessoas, deve ser incorporado à tag body. Essa tag vem depois da tag head.

DIV (<div></div>)

A tag div, também conhecida como a tag de divisão, é usada para definir a divisão do conteúdo no arquivo. Geralmente atua como contâiner para os outros elementos ou tags no arquivo. Isso geralmente é usado para estilizar o conteúdo do arquivo.

SPAN (<span></span>)

Ele é usado para envolver todo o texto no arquivo. É basicamente usado para agrupar o conteúdo embutido no arquivo.

A (<a href="https://www.google.com">Google</a>)

Essa é a tag de hiperlink. Nesta tag, você pode passar um link de página ou site para onde deseja direcionar o visualizador. Consiste no atributo "href" que é muito importante, pois o destino é passado por esse atributo.

BR (<br>)

É o comando usado para quebrar as linhas dentro do arquivo. Tudo o que ele faz é adicionar um espaçamento no parágrafo e garantir que a linha seguinte apareça na nova linha. Não há tag de fechamento para <br>.

IMG (<img src="https://bitarray.io/images.logo.png alt="bitarraylogo">)

Se você deseja colocar alguma imagem nos arquivos HTML, você pode fazer o mesmo usando a tag img. Você pode definir a altura e a largura da imagem usando outros atributos dentro dela e assim, pode personalizar a imagem de acordo com você. Não há tag de fechamento para <img>.

UL (<ul></ul>)

A tag é usada para definir a lista não ordenada no arquivo. Ele publica os limites com marcadores no site. Ele é um elemento de nível de bloco no qual os itens de lista são colocados.

Exemplo:

      <ul>Shopping list
        <li>Item 1
        <li>Item 2
        <li>Item 3
        <li>Item 4
      </ul>

LI (<li></li>)

A tag li é usada junto com a tag ul para passar a hierarquia da lista no arquivo. Quando você quer passar uma lista dentro de uma lista, usamos a tag li.

Exemplo:

      <ul>Books
        <li>fictional
        <li>classic
        <li>mystery
        <li>romance
      </ul>

TABELAS

Cada linha da tabela começa com um <tr> e termina com uma tag </tr>. Tudo entre <td> e </td> é o conteúdo da célula da tabela. Quando você quiser que suas células sejam cabeçalhos, nesses casos use a tag <th> em vez da <td>.

Exemplo:

    <table>
      <tr>
        <th>Roupas</th>
        <th>Farmácia</th>
        <th>Cosméticos</th>
      </tr>
      <tr>
        <td>Vestidos</td>
        <td>Remédios</td>
        <td>Rímel</td>
      </tr>
      <tr>
        <td>Meias</td>
        <td>Algodão</td>
        <td>Batons</td>
      </tr>
    </table>

FORMULÁRIOS

Formulários HTML são um dos principais pontos de interação entre um usuário e um web site ou aplicativo. Eles permitem que os usuários enviem dados para o web site. Um formulário HTML é feito de um ou mais widgets. Esses widgets podem ser campos de texto (linha única ou de várias linhas), caixas de seleção, botões, checkboxes ou radio buttons.
